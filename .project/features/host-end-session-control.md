The host can immediately end the entire session for all participants — a moderation kill-switch to shut things down if a participant contributes something offensive (e.g. an inappropriate display name, or offensive content once in-app authoring/drawing exists).
Why: safety/moderation control for a publicly-deployed, untrusted-traffic app; distinct from host-remove-participant (removes one person) — this terminates the whole session for everyone at once.
